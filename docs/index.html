rsvprdr

Read faster. Retain more.

rsvprdr is a Chrome extension that turns any selected text into an RSVP (Rapid Serial Visual Presentation) speed reading experience — one word at a time, with ORP letter highlighting to help your brain lock on faster.

How It Works

Select text on any webpage → right-click → "Speed Read Selection" → read at your pace (100–1000 WPM) with keyboard shortcuts.
